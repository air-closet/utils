# @air-closet/utils

Utility functions

## How to install

```bash
yarn add @air-closet/utils
```

## How to use

in typescript:

```ts
import { promise_all } from '@air-closet/utils'

await promise_all(promiseList);
```

in javascript:

```js
const { promise_all } = require('@air-closet/utils');

await promise_all(promiseList);
```

## List of exported functions

- `promise_all()`
