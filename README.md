# @air-closet/utils
Utility functions 

## How to install

wip

```bash
yarn add @air-closet/utils
``` 

## How to use

in typescript: 

```ts
import aircloset_utils from '@air-closet/utils'

await aircloset_utils(promiseList);
```

in javascript:

```js
const aircloset_utils = require('@air-closet/utils')

await aircloset_utils(promiseList);
```
